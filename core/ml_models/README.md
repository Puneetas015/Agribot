# core/ml_models/

This directory holds the trained ML model artifacts.
These files are generated by running the training scripts — they are NOT
committed to version control (add to .gitignore).

## Files created after training:

| File                  | Generated by                              | Size (approx) |
|-----------------------|-------------------------------------------|---------------|
| crop_rf_model.pkl     | python crop_recommendation/train_model.py | ~5 MB         |
| label_encoder.pkl     | python crop_recommendation/train_model.py | ~1 KB         |
| disease_model.h5      | python disease_detection/train_cnn.py     | ~15 MB        |

## How to generate them:

```bash
# Module 1 — Random Forest
python crop_recommendation/train_model.py

# Module 2 — MobileNetV2 CNN
python disease_detection/train_cnn.py
```

After training, this directory should contain:
  core/ml_models/crop_rf_model.pkl
  core/ml_models/label_encoder.pkl
  core/ml_models/disease_model.h5
